Privacy Policy

Last Updated: April 2026

1. Introduction

	This Privacy Policy explains how Heartrate2Pebble (the "Software") handles information. This Software is an open-source tool built for recreational and educational purposes. The core principle is simple: Your health data is yours alone.
	
2. Collection and Use of Heart Rate Data

    Real-Time Processing: The Software reads heart rate data from your connected wearable device solely to display it on your phone and pass it to your Pebble watch.

    No Storage: Heart rate data is processed in volatile memory (RAM) for real-time display. It is never saved to a database, log file, or permanent storage on your device.

    No Transmission: Data is never sent to any servers, third-party analytics, or external entities.

3. Local Configuration

	The Software may store minimal configuration data locally (such as a Bluetooth MAC address) to facilitate hardware connections.

    Local Only: This information stays strictly on your mobile device and is never synced to the cloud.

    Automatic Deletion: All configuration data is deleted by the Android system when the Software is uninstalled.

4. Open Source Transparency

	The source code for this Software is available for audit at https://github.com/meerkat42/heartrate2pebble. This transparency ensures that the Software functions exactly as described without hidden tracking or telemetry.

5. Disclaimer Alignment

	As stated in the Disclaimer, this Software is not a medical product. The developer provides this tool for recreational use and does not collect or manage your health information.
